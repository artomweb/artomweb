Blender-Spotify

Blender allows users to create and run Python scripts to generate 3D models. In this example we will use Blender do generate a keyring containing a Spotify Code.

The easiest way to understand Blender's Python commands is to open up the GUI into the Scripting workspace. Here, any modification made to Blender's UI is displayed in the console.

For example when deleting the default cube, this is the output to the console:

After importing Blender's module, bps in a new file, this command can be copied directly:

This line will delete the currently selected object, after selecting an object in the viewport. To run the script press the right pointing triangle at the top of the editor.

Creating the base

To create the keyring we will start with a cube as a base and subtract from it a cylinder for the keyring hole.

First we will delete the default objects by selecting and deleting everything:

bpy.ops.object.select_all(action='SELECT')
bpy.ops.object.delete()
bpy.ops.object.select_all(action='DESELECT')

To add a cube and scale it:

bpy.ops.mesh.primitive_cube_add(enter_editmode=False, align='WORLD', location=(0, 0, 0))
bpy.context.object.scale = (0.005, 0.03, 0.001)

To add a cylinder:

bpy.ops.mesh.primitive_cylinder_add(radius=0.003, depth=0.005, enter_editmode=False, align='WORLD', location=(0, -0.024, 0))

Next we can assign the cube and cylinder to their own variables, so that they are easy to reference:

cube = bpy.data.objects['Cube']
cylinder = bpy.data.objects['Cylinder']

Running the above code will give us this result:

We can then use a boolean modifier to subtract the cylinder from the cube:

bool = cube.modifiers.new(type="BOOLEAN", name="bool")
bool.object = cylinder
bool.operation = 'DIFFERENCE'
bpy.context.view_layer.objects.active = cube
bpy.ops.object.modifier_apply(modifier="bool")

We can now delete the cylinder to avoid confusion later when exporting:

bpy.ops.object.select_all(action='DESELECT')
cylinder.select_set(True)
bpy.ops.object.delete()

Adding the Spotify code

With the base of the keyring finished we can now add the Spotify code. The requests module can be used to download the code as an SVG

import requests

URI = input("Enter a song URI")

r = requests.get("https://scannables.scdn.co/uri/plain/svg/ffffff/black/640/" + URI, stream=True)

with open("SpotifyCodeDownload.svg", 'wb') as f:
    for chunk in r.iter_content(1024): 
        f.write(chunk)

The code will be saved in the directory that the script is being run in and so it can be imported into Blender:

bpy.ops.import_curve.svg(filepath="SpotifyCodeDownload.svg", filter_glob="*.svg")

The SVG imports as a collection of curves:

All of the curves are needed except from 'Curve' which is a rectangle around the code, so this can be deleted:

bpy.ops.object.select_all(action='DESELECT')
bpy.data.objects['Curve'].select_set(True)
bpy.ops.object.delete()

Next we need to join the curves together and then convert them into a mesh so that they can be extruded. Firstly the collection is selected however there is now no active object so 'Curve.001' is set as the active object. Next the curves are converted and the origin is recalculated.

for obj in bpy.data.collections['SpotifyCodeDownload.svg'].all_objects:
        obj.select_set(True)


bpy.context.view_layer.objects.active = bpy.data.objects["Curve.001"]

bpy.ops.object.join()

bpy.ops.object.convert(target='MESH')

bpy.ops.object.origin_set(type='ORIGIN_GEOMETRY', center='MEDIAN')

The new mesh can now be moved into the correct position, these values can only be found by manually moving the object in the viewport however they will be the same for every code:

curve = bpy.data.objects["Curve.001"]

bpy.context.object.rotation_euler[2] = 1.5708
bpy.ops.transform.resize(value=(0.277968, 0.277968, 0.277968))


bpy.ops.transform.translate(value=(-0.0734364, -0.0216113, 0.0009043))

Finally the mesh is extruded in edit mode:

bpy.ops.object.mode_set(mode='EDIT')

bpy.ops.mesh.select_all(action='SELECT')

bpy.ops.mesh.extrude_region_move(MESH_OT_extrude_region={"use_normal_flip":False, "mirror":False}, TRANSFORM_OT_translate={"value":(0, 0, 0.000875622), "orient_type":'NORMAL', "orient_matrix":((-0.538016, -0.842934, 0), (0.842934, -0.538016, 0), (0, 0, 1)), "orient_matrix_type":'NORMAL', "constraint_axis":(False, False, True), "mirror":False, "use_proportional_edit":False, "proportional_edit_falloff":'SMOOTH', "proportional_size":1, "use_proportional_connected":False, "use_proportional_projected":False, "snap":False, "snap_target":'CLOSEST', "snap_point":(0, 0, 0), "snap_align":False, "snap_normal":(0, 0, 0), "gpencil_strokes":False, "cursor_transform":False, "texture_space":False, "remove_on_cancel":False, "release_confirm":False, "use_accurate":False})

bpy.ops.object.mode_set(mode='OBJECT')

Exporting

All of the objects are now selected and exported:

bpy.ops.object.select_all(action='SELECT')
bpy.ops.export_mesh.stl(filepath="out.stl", use_selection=True, global_scale=1000)

The code can be run with this command:

blender -b --python BlenderStl.py
