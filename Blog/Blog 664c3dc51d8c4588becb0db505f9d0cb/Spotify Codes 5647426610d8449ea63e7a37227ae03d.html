Spotify Codes

This is a Spotify Code, it can be scanned by the Spotify app to take you directly to a song, playlist or album. The app can scan the code incredibly quickly, even if the imaged is skewed or contains different colours.

Generating Codes

In the app the code appears under the album art for a specific song. The code along with the album cover can be saved as an image. To generate codes online Spotify has a page called https://www.spotifycodes.com/. This allows users to save codes in any size, format or colour. By looking at the network tab of the chrome developer tools we can see that the website makes a request to https://scannables.scdn.co/uri/plain/[format]/[background-color-in-hex]/[code-color-in-text]/[size]/[spotify-URI] containing the before mentioned parameters. This site can generate codes in jpg, png or svg formats.

This allows us to use Python to download a code for a given URI.

import requests

URI = input('Enter a uri')

r = requests.get("https://scannables.scdn.co/uri/plain/svg/ffffff/black/640/" + URI, stream=True)

if r.status_code == 200:

    with open("SpotifyCodeDownload.svg", 'wb') as f:
        for chunk in r.iter_content(1024): 
            f.write(chunk)
else:
    print('Not a valid uri')

By opening the svg in a text editor we can begin to understand how these patterns work. This is an extract from the SVG of a code:

<rect x="100.00" y="44.50" width="6.71" height="11.00" rx="3.36" ry="3.36" fill="#ffffff"/>
<rect x="112.42" y="23.50" width="6.71" height="53.00" rx="3.36" ry="3.36" fill="#ffffff"/>
<rect x="124.84" y="44.50" width="6.71" height="11.00" rx="3.36" ry="3.36" fill="#ffffff"/>
<rect x="137.27" y="37.50" width="6.71" height="25.00" rx="3.36" ry="3.36" fill="#ffffff"/>

Each line draws a bar with a height that varies from 60.00 to 11.00.

A Python script can be used to convert the SVG into a string of values between 0 and 7, this is much easier than doing an analysis of the jpg image:

import xml.dom.minidom

doc = xml.dom.minidom.parse('spcode.svg')

heights = {'11.00': 0, '18.00': 1, '25.00': 2, '32.00': 3, '39.00': 4, '46.00': 5, '53.00': 6, '60.00': 7}

output = ''

for bar in doc.getElementsByTagName('rect')[1:]:
    output += str(heights[bar.attributes['height'].value])

print(output)

By looking at these three codes we can also see that the first and last bars are always at a height of 0 and the middle bar always has a height of 7, the maximum and minimum.
